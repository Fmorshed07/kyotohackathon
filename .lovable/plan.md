

# IMPACT TOKYO 2026 – Website Experience Plan

## The Vision
A cinematic, high-end event website that feels like an **Apple product launch meets GTA-6 title sequence**, set in futuristic Tokyo. Minimal, smooth, powerful, and quietly confident.

---

## 1. Cinematic Loading Experience
When the site first opens:
- **Black screen** holds for a moment
- A **thin cyan line draws horizontally** across the screen
- **"IMPACT TOKYO"** fades in with a subtle glow
- **"AI FOR GLOBAL GOOD"** appears below
- The entire screen **gently slides/reveals** into the main experience
- Total duration: ~3 seconds of controlled suspense

---

## 2. Visual Design System
**Color Palette:**
- Deep navy & charcoal base
- Subtle cyan and navy accent highlights
- No photos, no illustrations – only gradients, light, typography, and motion

**Typography:**
- Large letter-spaced headlines (GTA title card style)
- Small, clean body text (Apple UI style)
- Google Fonts: Space Grotesk for titles, Inter for body

---

## 3. Hero Section (Full Viewport)
- Centered, breathing text:
  - **"IMPACT TOKYO"** (large, gradient cyan-to-white)
- **"AI FOR GLOBAL GOOD"** (below, navy accent)
  - *"Build With Purpose. Code With Vision. Impact the World."*
  - "Organized by Cognisor AI"
- Two glowing CTA buttons:
  - **"Become a Sponsor"** → External link
  - **"Join the Hackathon"** → External link
- Subtle animated gradient background with slow movement

---

## 4. Scroll Experience
Apple-style smooth scrolling with:
- **Parallax effects** on section reveals
- **Sections fade and slide in** softly
- Slow easing for a luxurious feel
- Each section occupies its own "moment"

---

## 5. Sections Breakdown

### About Section
Large cinematic text blocks explaining:
- "Impact > Hype" philosophy
- Problem-first, impact-driven approach
- Built in Tokyo for the world

3. Education, Skills & Workforce
4. Human-Centered AI & Ethics
5. Open Impact (Wildcard)

Each track with a one-line description

- Location: "Tokyo, Japan" (venue TBA)
- In-person format

- Students & Researchers

---

## 6. Page Transitions & Animations
- **Fade through black** between major sections
- Subtle **blur transitions**
- Light sweep effects
- No hard cuts – everything flows like a film camera
- Elements fade in with slight upward motion

---

## 7. Mobile Experience
Fully responsive with:
- Same cinematic quality on mobile
- Vertical flow with Apple-website elegance
- Touch-friendly large buttons
- Same calm, luxurious animations
- Hamburger menu if needed for navigation

---

## 8. Technical Approach
- Single-page React application with smooth scroll
- CSS animations and Intersection Observer for scroll-triggered reveals
- Custom animated gradient backgrounds
- No external images needed – pure CSS/SVG gradients and typography

---

## Final Emotion
Visitors will feel:
*"This is not just an event. This is a global technology movement."*
